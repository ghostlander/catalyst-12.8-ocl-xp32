This ATI/AMD Radeon driver package supports Windows XP SP3 32-bit and Windows
Server 2003 SP2 32-bit. It consists of the following parts:

- 2D and Direct3D driver pulled out of Catalyst 12.8;
- DXVA driver pulled out of Catalyst 12.6;
- OpenGL driver pulled out of Catalyst 12.3;
- CAL libraries v1.4.1683 (17-May-2012);
- OpenCL 1.2 support by AMD APP SDK v2.7 (938.2);
- AVIVO encoder and transcoder v11.7.0.11130 (30-Nov-2011).
